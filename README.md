Evidencia de aprendizaje. Programa con sockets

Como parte de la evaluación de esta unidad, es necesario que realices un programa en JAVA 
que permita el intercambio de información entre dos computadoras para lograr comunicación 
entre los usuarios de estas (chat).

Propósito: Crear un programa que comunique dos computadoras y permita comunicación entre 
los usuarios de estas (chat).

Instrucciones:

1.  Identifica los componentes y la estructura requerida para el manejo de sockets.

2.  Desarrolla la interfaz gráfica del programa, esta deberá tener componentes gráficos
necesarios para dar coherencia gráfica al programa.

3.  Desarrolla los métodos que proporcionen la funcionalidad requerida al programa.
